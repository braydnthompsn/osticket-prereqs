<h1>osTicket - Prerequisites and Installation</h1>
Here, I'll go over the installation process of osTicket, an open source ticketing system.<br />

<h2> Software & Technology Used</h2>

- Windows 10 
- Microsoft Azure (Virtual Machines)
- Remote Desktop (RDP)
- Internet Information Services (IIS)

  <h2> Prerequisites </h2>

- Create a Virtual Machine in Azure
- Install osTicket
- Install HeidiSQL
- Install MySQL
- Install PHP
- Install Microsoft Visual C++ Redistributable
  <h2>Steps</h2>
<h3 align="center">Create Virtual Machine in Azure</h3>
<br />
<p>
-To start, create a resource Group inside of the Azure Portal.
<br />
</p>
<p>
	<img src="https://i.imgur.com/eBi5k2l.png" height="75%" width="100%" />
</p>
<p>
-Make a Windows 10 VM with preferrably 2-4 Virtual CPUs. For your username and password, just make it something you will remember. When creating the Virtual Machine (VM), allow Azure to create a new Virtual Network (Vnet):</h3>
<br />
</p>
<p>
	<img src="https://i.imgur.com/dEF1c7h.png" height="75%" width="100%" />
